CSS translate 예제

transform: translate는 요소를 x, y축 방향으로 시각적으로 이동시키는 CSS 속성.

  • translateX(px): X축(가로)으로 이동
  • translateY(px): Y축(세로)으로 이동
  • translate(x, y): X, Y축 동시에 이동
  • 원리: translate는 요소의 레이아웃 위치는 그대로 두고, 화면에만 이동된 것처럼 보이게 만듭니다.
  • 주의: translate로 이동한 요소는 실제로는 원래 위치에 있으므로, 다른 요소와의 겹침이나 이벤트(클릭 등) 처리에 영향을 줄 수 있습니다.
  • 활용 팁: 애니메이션, 드래그&드롭, 인터랙션, 모달/팝업 등장 효과 등에 자주 사용됩니다.
  • 성능: transform은 GPU 가속이 적용되어 부드러운 애니메이션이 가능합니다.
  • 조합: scale, rotate 등과 함께 사용하면 다양한 시각 효과를 연출할 수 있습니다.
