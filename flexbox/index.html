CSS Flexbox 예제

Flexbox는 1차원 레이아웃을 쉽게 만들 수 있는 CSS3의 레이아웃 모델.
컨테이너에 display: flex를 지정하면, 자식 요소들이 자동으로 flex 아이템이 되어 다양한 정렬과 배치가 가능.

  • display: flex - flex 컨테이너를 만듭니다.
  • flex-direction - 아이템의 배치 방향을 지정합니다. (row, column 등)
  • justify-content - 주축(가로/세로)의 정렬 방식을 지정합니다.
  • align-items - 교차축(세로/가로)의 정렬 방식을 지정합니다.
  • flex-wrap - 아이템이 컨테이너를 넘칠 때 줄바꿈 여부를 지정합니다.
  • gap - 아이템 사이의 간격을 지정합니다.
