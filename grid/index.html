CSS Grid

CSS Grid는 2차원 레이아웃 시스템으로, 행(row)과 열(column)을 동시에 제어.

Flexbox가 1차원(한 방향)이라면, Grid는 2차원(행과 열 모두)을 다룰 수 있어 복잡한 레이아웃을 쉽게 만들 수 있습니다.

1. 기본 Grid (3열 레이아웃)

grid-template-columns: repeat(3, 1fr)

✅ 핵심 개념: 3개의 열을 만들고, 각 열이 동일한 너비(1fr)를 가집니다.

📝 fr 단위: fraction의 줄임말로, 사용 가능한 공간을 비율로 나눕니다.

2. Grid Rows (행 높이 제어)

grid-template-rows: 60px 60px 60px

✅ 핵심 개념: 각 행의 높이를 명시적으로 지정할 수 있습니다.

📝 활용법: 헤더, 메인, 푸터 영역의 높이를 다르게 설정할 때 유용합니다.

3. Grid Gap (간격 조절)

gap: 24px

✅ 핵심 개념: 그리드 아이템 사이의 간격을 한 번에 설정합니다.

📝 세부 설정: row-gap, column-gap으로 행과 열의 간격을 따로 설정할 수도 있습니다.

4. Grid Area (영역 기반 레이아웃)

grid-template-areas로 레이아웃을 시각적으로 정의

✅ 핵심 개념: 영역에 이름을 부여하고, 각 요소를 해당 영역에 배치합니다.

📝 장점: 레이아웃 구조를 한눈에 파악할 수 있고, 반응형 디자인에서 쉽게 재배치 가능합니다.

5. Grid 라인 기반 배치

grid-column: 1 / 3, grid-row: 1 / 2

✅ 핵심 개념: 그리드 라인 번호를 사용해 정확한 위치에 아이템을 배치합니다.

📝 활용법: 특정 아이템을 여러 셀에 걸쳐 배치할 때 사용합니다.

6. 자동 배치 (Auto-fit)

grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))

✅ 핵심 개념: 화면 크기에 따라 자동으로 열의 개수가 조정됩니다.

📝 반응형: 브라우저 창 크기를 조정해보세요! 자동으로 레이아웃이 변경됩니다.

7. 정렬 속성

justify-items, align-items, place-items

✅ 핵심 개념: 그리드 아이템들을 셀 내에서 정렬하는 방법을 제어합니다.

📝 차이점: justify는 수평, align은 수직, place는 둘 다 한번에 설정합니다.

🎯 CSS Grid 핵심 요약

  • display: grid - 그리드 컨테이너 생성
  • grid-template-columns/rows - 열/행 크기 정의
  • gap - 아이템 간 간격 설정
  • grid-template-areas - 영역 기반 레이아웃
  • grid-column/row - 라인 기반 배치
  • auto-fit/auto-fill - 반응형 레이아웃
